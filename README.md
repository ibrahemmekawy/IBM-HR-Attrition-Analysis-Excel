# HR Attrition Risk Dashboard (Excel)

An interactive Excel dashboard that analyzes employee attrition and identifies key factors driving workforce turnover. The project provides data-driven HR insights to support better retention strategies and organizational planning.

---

## ✅ Features
- Overall attrition metrics (total employees, active employees, attrition rate)
- Department-wise and job-level attrition breakdown
- Gender and age group analysis
- Impact of salary hike, monthly income, and job satisfaction
- Interactive slicers and filters for dynamic exploration

---

## ✅ Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Interactive Filtering
- Data Cleaning & Visualization

---

## ✅ How to Use
1. Download the Excel file from the repository.
2. Open it in Microsoft Excel.
3. Use the filters and slicers to explore attrition insights.
4. Navigate through charts to analyze trends and patterns.

---

## ✅ Dataset
- Contains employee demographics, salary, job role, satisfaction, travel frequency, and attrition status.
- Used for exploratory HR analytics and dashboard visualization.

---

## ✅ Future Enhancements (Optional)
- Add predictive analysis for attrition risk
- Compare trends across different time periods
- Automate data refresh

---

## ✅ Project Purpose
To help HR teams understand why employees leave, identify high-risk areas, and support better decision-making through visual analytics.
